Question-1: What are the different ways to select an element in the DOM?


There are many built in methods in JavaScript to select an element in the DOM .

  • By ID => getElementById()
  • By Class Name => getElementsByClassName()
  • By Tag Name => getElementsByTagName()
  • Query Selector => querySelector()
  • Query Selector All => querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?


The main difference between innerHTML, innerText and textContent is ->

"innerHTML" can contain the complete HTML content within an element, while "innerText" can contain only the visible text content of an element and ""textContent"" can contain all the text of an element ,including hidden text and whitespace, but also ignoring HTML tags.

Question-3: What is event delegation in the DOM?


In DOM :
"Event delegation" is a technique in JavaScript where programmer can add a single event listener to a parent element instead of multiple listeners to child elements .

Question-4: What is event bubbling in the DOM?


Event bubbling in JavaScript is a mechanism in the DOM where an event starts from the target elements and then propagates (bubbles) upward through its ancestors .

Question-5: How do you create, add, and remove elements using JavaScript?


There are many built in methods in JavaScript to create, add adn remove elements dynamically in the DOM .

  • By using document.createElement() - method we can create new elements
  • By using appendChils(), append(), prepend(), insertBefore(), innerHTML() - methods are used to generate or add elements
  • By using removeChild(), remove() - methods are used to remove elements
